Welcome to JQualityVis

JQualityVis is a software quality analysis framework that allows engineers to visualise the results of static software metrics on parsed source code.

Analyse the source code of any Java project and apply all of the metrics in the Bansiya and Davis QMOOD Quality Model, amongst others.

View graphical representations of the quantitative and relational measurements in forms such as a scatter graph, charts and a treemap.
